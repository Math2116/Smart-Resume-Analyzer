About Smart Resume Analyzer

Smart Resume Analyzer evaluates resumes against job descriptions with ATS scoring, skills mapping, and AI suggestions.

Tools & Technologies
  • Frontend: HTML, CSS, JavaScript, Bootstrap
  • Backend: Java (Spring Boot, Maven)
  • Text Extraction: Apache Tika
  • NLP & AI: Apache OpenNLP, OpenAI API
  • Database: SQLite (JDBC)
  • Tools: IntelliJ IDEA, GitHub, Log4j2, JUnit 5, Docker

Purpose: help candidates understand job fit and improve resumes with clear, actionable feedback.
